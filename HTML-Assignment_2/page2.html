Attributes

The HTML attributes are used to provide more information to an HTML Element.

  • HTML attributes only appears at start tags. It will never be on end tag.
  • All HTML elements can have attributes.
  • HTML elements can have multiple attributes.
  • HTML attributes usually come in name/value pair like:name="value"
  • An attribute name
  • An equal =sign
  • A value surrounded by qoutation marks "value"

You can use single qoutation marks depending on the situation esp. When the value contains double qoutes.

There are two ways to specify the URL in the href attributes

  1. Absolute URL:Links to add an external website that is hosted on another server.
  2. Relative URL:Links to an html page that is hosted with the website. Here, the URL does not inculude the domain name. If the URL begins without a slash, it will bbe relative to the current page. Example: href="login.html
